# V1 已知限制

- 当前后端上传接口没有服务端幂等键保证，V1 不宣称 exactly-once；响应未知时进入 `uncertain` 并要求人工核对。
- 单 EXE 免安装，但依赖目标机器已有 Microsoft Edge WebView2 Evergreen Runtime。
- V1 验收保证四路并发；配置预留至八路不代表八路已通过性能和稳定性验收。
- 物理串口识别受驱动和 USB 转串口芯片信息质量影响，VID/PID 相同且无唯一序列号时仍需绑定物理位置。
- 磁盘阈值不能替代操作系统级容量监控；禁止自动删除未上传文件。
- 8 小时和 24 小时结果仅对报告中记录的吞吐、平均行长、上传频率、设备和 Windows 环境有效。
- 72 小时验证在发布后的试点工位完成，不是候选版本生成的阻断条件。
- 在线更新、自动回滚、云端配置同步、复杂异常面板、上下文提取和时间窗口去重不属于 V1。
- WebView2 缺失、Win10 1809、较新 Win10 和 Win11 均需在真实目标环境人工签字确认。
